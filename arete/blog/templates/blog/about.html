What is CodeArete?

WIP Code Projects
I will be posting blog WIP projects on the website as I go through my journey as a web developer/software developer. These posts will detail my thoughts on my application, why I’m building it and the problems I’ve come to beat and problems I think that will arrive. I will be going as detailed as possible, discussing algorithms, clean code and future features that will be added.
Active Blog for technology trends
CodeArete will be updated once a week with posts on the various technology topics of Programming projects I’m working on, things going on in the technology industry, Ideas for future projects for beginners, Write ups about languages that should be learned in the industry and what they should be used for.
Website constantly updated
Features will be added to CodeArete website as my skills are developed, the website will be updated to look better, go through several revamps and be thoroughly documented to showcase my skills increasing. Features such as comments, user login and email newsletters will be added as time passes and skills get better. The website will also have a portfolio section added to showcase completed projects to potential employers.
Portfolio
CodeArete at its core will be a blog but a portfolio section will be added soon to showcase completed projects. These projects will have detailed posts about the idea behind the project, the features, and the design of it. I will walk through all the steps and logic of each project. Detailed comments of what each function does will be provided with source uploaded at Github (All will be open source with links to my personal GitHub)
